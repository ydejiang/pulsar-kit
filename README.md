# pulsar-kit

A toolkit for pulsar hunters.

This repository contains tools for pulsar data analysis and aims to share useful utilities with the pulsar astronomy community.

Happy pulsar hunting!

##
1. `residuals_plot.py`, plot pulsar timing residuals from TEMPO resid2.tmp file.
2. `ACCEL_sift.py`, this is a modified version of `ACCEL_sift.py` from PRESTO, offering greater flexibility by allowing users to specify custom input parameters.
